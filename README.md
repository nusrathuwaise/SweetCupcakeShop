SweetCupcakeShop-OOP

A Java-based cupcake shop management system demonstrating Object-Oriented Programming (OOP) concepts, developed in NetBeans.

About the Project

SweetCupcakeShop allows users to manage cupcakes, process orders, and track inventory using OOP principles like classes, objects, and inheritance.

Features

Add, view, update, and delete cupcake products

Process customer orders

Track inventory

How to Run

Open the project in NetBeans IDE.

Run the project using the Run button.

Use the menu to manage products and orders.
